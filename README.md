# VulHub Leaderboard - Clean & Simple

**Purpose:** Track student submissions of VulHub vulnerability exploits

## What This Does

Students exploit vulnerabilities from https://github.com/vulhub/vulhub, then submit:
- Screenshot of successful exploit
- Which VulHub vulnerability they exploited
- Notes about their findings

The leaderboard ranks students by successful submissions and awards badges.

## Features

- User registration/login
- Submit exploit proof (screenshot + notes)
- Live leaderboard
- Automatic badge awards
- Simple, clean, working

## Deploy Time

- API to Heroku: 5 minutes
- Frontend to Vercel: 5 minutes
- **Total: 10 minutes**

## Stack

- API: NestJS + PostgreSQL + Prisma
- Frontend: Next.js + Tailwind
- Database: Heroku PostgreSQL (already configured)
